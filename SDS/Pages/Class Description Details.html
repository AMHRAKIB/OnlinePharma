4.0. Class Description Details

User: User class is an abstract class which contains the data of a user for example name, id, password ,email. It also contains methods which can update user data.

Admin: Admin is a class which inherited user class. It contains the data of an admin for example name,email.Admin will also be able to check information,update information, delete account, block account etc. Parent is a class which inherited user class. It contains the data of a user.

Cart: Cart class contains the data of buying medicine, information of quantity, medicine name, generic name,total price.

Client: Client is a interface of a user. Thisclass contains client name, address, email, buying information. There is few methods which are used by clients to register, login& update profile.

Order: This Class contains data of client how many orders has been done. This class shows the status of the client. The quantity of medicine the client has ordered.This class contains the order details of medicines.

Medicine: This Class contains data of the name, generic name, medicine description, production date & expiry date of the medicine. There are few methods in this class add,delete,update medicine which is maintained by admin.

Prescription: In this class user will be able upload their prescription. There will be a method in this class which is add prescription.

Search: This class contains a search option where user will be able to search their medicine by their generic name.
